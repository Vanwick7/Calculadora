# 🧮 Calculadora

Uma calculadora simples e funcional desenvolvida com HTML, CSS e JavaScript puro.

## 📸 Preview

> Adicione uma screenshot do projeto aqui!

## 🚀 Funcionalidades

- ➕ Adição
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 🔄 Limpar cálculo
- 📟 Interface simples e intuitiva

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura da página
- **CSS3** — Estilização e layout
- **JavaScript** — Lógica e funcionalidades

## 📁 Estrutura do Projeto

```
Calculadora/
│
├── index.html   # Estrutura da calculadora
├── style.css    # Estilos visuais
└── script.js    # Lógica de funcionamento
```

## ▶️ Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/Vanwick7/Calculadora.git
```

2. Abra o arquivo `index.html` no seu navegador

3. Pronto! É só calcular 🎉
